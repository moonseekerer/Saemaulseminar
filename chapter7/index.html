💡 죄수의 딜레마(Prisoner's Dilemma)란?
쌍방 배신(+1)보다 쌍방 협력(+3)을 하는 것이 윈윈(Win-Win)입니다. 하지만 내가 협력할 때 상대가 배신하면 나는 0점, 상대는 5점을 얻습니다. 상대방의 배신이 두려워서, 혹은 상대방을 속여 큰 이익(+5)을 얻고 싶은 유혹 때문에 결국 둘 다 배신(+1)을 선택하게 되는 모순적 상황을 의미합니다.

맞대응전략의 수학적 분석 (3라운드)

A가 맞대응전략을 쓸 때, B가 협력(C)을 선택할 유인 탐구

전체 보수  =  (1라운드의 보수)  +  𝛽 (2라운드의 보수)  +  𝛽² (3라운드의 보수)

* 𝛽 (미래할인인자) : 0 < 𝛽 < 1 의 값을 가집니다. 미래의 효용을 현재가치로 환산하며, 값이 클수록 미래를 더 중시한다는 의미입니다.

1. B의 선택 전략 조합

3라운드가 끝나면 더 이상 게임이 진행되지 않으므로, B는 마지막 3라운드에서 반드시 우월전략인 배신(D)을 선택합니다. 문제는 미래가 남은 1, 2라운드의 선택입니다. B가 취할 수 있는 선택 조합과 결과는 다음과 같습니다:

  • ✅ CCD 조합: 1R(C), 2R(C), 3R(D)   ➡   𝑅 + 𝛽 𝑅 + 𝛽² 𝑇
  • ✅ DCD 조합: 1R(D), 2R(C), 3R(D)   ➡   𝑇 + 𝛽 𝑆 + 𝛽² 𝑇
  • ✅ CDD 조합: 1R(C), 2R(D), 3R(D)   ➡   𝑅 + 𝛽 𝑇 + 𝛽² 𝑃
  • ✅ DDD 조합: 1R(D), 2R(D), 3R(D)   ➡   𝑇 + 𝛽 𝑃 + 𝛽² 𝑃

2. 협력이 합리적일 조건 (CCD가 최적일 조건)

B가 1라운드와 2라운드에서 C(협력)를 선택하려면, CCD 조합의 전체보수가 나머지 경우보다 커야 합니다.

  1. CCD > DCD: 𝑅 + 𝛽 𝑅 + 𝛽² 𝑇  >  𝑇 + 𝛽 𝑆 + 𝛽² 𝑇
     ➡  정리하면 𝛽  >  (𝑇 - 𝑅) / (𝑅 - 𝑆)
  2. CCD > CDD: 𝑅 + 𝛽 𝑅 + 𝛽² 𝑇  >  𝑅 + 𝛽 𝑇 + 𝛽² 𝑃
     ➡  정리하면 𝛽  >  (𝑇 - 𝑅) / (𝑇 - 𝑃)
  3. CCD > DDD: 부등식을 풀면 앞의 두 조건과 일치함을 알 수 있습니다.

∴ 결론:   𝛽  >  max { (𝑇-𝑅) / (𝑅-𝑆) , (𝑇-𝑅) / (𝑇-𝑃) }   일 때 B는 늘 협력합니다.

🔎 이론 인터랙티브 시뮬레이터

위의 수학적 조건을 직접 변수(T, R, P, S, 𝛽)를 조절하며 테스트해보세요. 𝛽의 변화에 따라 4가지 선택 조합(CCD, DCD, CDD, DDD) 중 어떤 조합이 가장 높은 보수를 가져다주는지 실시간으로 확인할 수 있습니다.

조합별 전체 보수 결과:

3. 협력(새마을운동) 성공 조건을 만드는 요인

위 수식이 보여주는 조건은, 특정 환경 아래에서 맞대응전략이 어떻게 협력을 유도하는지명증합니다.

1. 미래할인인자(𝛽)가 클수록 📈

𝛽가 클수록 위 조건이 아주 쉽게 충족됩니다. 새마을운동 당시 '후손에게 가난을 물려줄 수 없다'는 의식이 확산되었는데, 이는 미래를 중시하는 성향(𝛽 상승)으로 해석되며 정부-주민의 협력 가능성을 대폭 높였습니다.

2. 상호 협력 보수(R)가 높을수록 🎁

R이 클수록 협력 유인이 강화됩니다. 우수 마을에 대한 자재 지원, 하사금, 마을 비석, 지도자 초청 등 다양한 보상적 조치는 협력의 보수(R)를 높여 성공적으로 협력을 이끌어내는 인센티브 설계였습니다.

3. 배신의 이익(T)이 낮을수록 📉

T가 작을수록 상대방을 배신할 유혹이 지워집니다. 수식의 조건을 T에 대해 미분하면 양수 (𝑅-𝑃) / (𝑇-𝑃)² 가 나오며, 이는 나 혼자 얌체처럼 이익을 독식(T)하는 게 어려워질수록 오히려 협력이 한층 강하게 유도됨을 수학적으로 증명합니다.
